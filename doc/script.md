# Script

* Hello my name is Ammon Phipps and this is my final project
* IT is called astrophotgraphy session Manager and its an application designed to help astrophotography hobbyists like myself document and share their sessions, photos, equipment and settings information with a community of like-minded individuals.

## Features

* Features for this application include user authentication, session creation with detailed records of astrophotography sessions including title, target, date and time, location, imaging statistics, caption and description and public/private visibility option. My favorite feature is the integration with the astronomy API for target name lookup, and the ability to share photos with the community.

* Lets start by creating an account. 

* After logging in, we are taken to the home page which displays a list of our sessions. From here we can create a new session, and view our existing sessions. The navigation bar at the top allows us to access other pages such as the public feed, liked sessions, and profile page.
